# AI Prompts - Motivation & Dopamine Hacks

## Overview
These prompts help when you lack motivation, need a boost, or want to tap into dopamine-friendly strategies to get things done.

## Prompt 1: Instant Motivation Boost

### When to Use
- Zero motivation right now
- Need energy and encouragement
- Feeling stuck in low-drive state

### The Prompt
```
I have zero motivation right now, but I need to do: [TASK]

Please give me:
1. A pep talk that understands ADHD challenges (not toxic positivity)
2. A visualization of how good I'll feel when it's done
3. A suggestion for a dopamine reward after completion
4. Permission to do a "good enough" version instead of perfect

Current energy level: [1-10]
Why I think I'm low motivation: ___
What usually helps me (if known): ___
```

### Example Usage
```
I have zero motivation right now, but I need to do: Finish the project report

Please give me:
1. A pep talk that understands ADHD challenges (not toxic positivity)
2. A visualization of how good I'll feel when it's done
3. A suggestion for a dopamine reward after completion
4. Permission to do a "good enough" version instead of perfect

Current energy level: 3/10
Why I think I'm low motivation: Feeling overwhelmed by perfectionism
What usually helps me: Small wins and clear next steps
```

### Copy-Ready Version (Minimal)
```
Zero motivation for: [TASK]
Energy: [X/10]
Give me: ADHD-friendly pep talk, visualization, dopamine reward, permission to do "good enough"
```

## Prompt 2: Hyperfocus Starter

### When to Use
- Want to enter hyperfocus mode intentionally
- Have a task worthy of deep work
- Need to maximize focus time

### The Prompt
```
I want to enter hyperfocus mode for: [TASK/PROJECT]

Help me:
- Create a compelling "mission statement" for this work session (make it exciting!)
- List what I need to prepare before I start (so I don't break flow)
- Set a realistic time goal (30-90 min blocks work best)
- Design a transition ritual for when I'm done
- Remind me to set a timer so I don't forget to eat/move

Why this task is important: ___
Potential distractions: ___
Time I have available: ___
```

### Example Usage
```
I want to enter hyperfocus mode for: Writing my book chapter

Help me:

- Remind me to set a timer so I don't forget to eat/move

Why this task is important: This is my passion project and I want to make progress
Potential distractions: Phone notifications, family interruptions
Time I have available: 2 hours
```

### Copy-Ready Version (Minimal)

- Would benefit from someone "there"

- Give gentle encouragement without judgment
- Help me get back on track if I've wandered
- Celebrate progress at each check-in

My planned work time: ___
What usually distracts me: ___

## Prompt 5: Energy Matching

### When to Use
- Not sure what to work on
- Energy level doesn't match planned task
- Need suggestions for current state

### The Prompt
```
My energy level right now is: [LOW/MEDIUM/HIGH]
My available time is: [MINUTES]

Here's my task list:
[PASTE YOUR TASKS]

Based on my current state, suggest:
- 3 tasks that match this energy level
- What I should definitely NOT try to do right now
- A 5-minute reset activity if I'm too low to do anything
- How to use this energy level wisely
```

### Example Usage
```
My energy level right now is: LOW (2/10)
My available time is: 45 minutes

Here's my task list:
- Write detailed project proposal (HIGH energy)
- Respond to simple emails (LOW energy)
- Organize desk (MEDIUM energy)
- Research for presentation (HIGH energy)
- File receipts (LOW energy)
- Call client (MEDIUM energy)

Based on my current state, suggest:
- 3 tasks that match this energy level
- What I should definitely NOT try to do right now
- A 5-minute reset activity if I'm too low to do anything
- How to use this energy level wisely
```

### Copy-Ready Version (Minimal)
```
Energy: [X/10], Time: [MINUTES]
Tasks: [LIST]
Match tasks to energy + tell me what NOT to do + reset activity if needed
```

## Prompt 8: Competition Creator

### When to Use
- Bored by routine task
- Want to gamify something
- Need artificial stakes for motivation

### The Prompt
```
I need to do this routine task: [TASK]
It's boring but necessary.

Create a game/competition for me:
- Challenge myself (beat my previous time/score)
- Create artificial stakes (what happens if I win/lose)
- Design a points/scoring system
- Add elements of surprise or variety
- Make it social (compete with or share with others)

My competitive nature: [VERY/SOMEWHAT/NOT REALLY]
What motivates me: [achievements, helping others, improving, etc.]
```

## Prompt 9: Future Self Motivation

### When to Use
- Present you doesn't care
- Need perspective shift
- Want to connect to bigger picture

### The Prompt
```
I don't want to do this: [TASK]

Help me connect with my future self:
- How will Future Me (1 hour from now) feel if I do this?
- How will Future Me (tomorrow/next week) benefit?
- What will Future Me be grateful that Present Me did?
- What is Present Me stealing from Future Me by not doing this?
- Remind me that I'm doing this FOR myself, not TO myself

Why I'm resisting: ___
What I actually want: ___
```

## Building Your Motivation System

### Create a Motivation Playbook
1. **Track what works**: Note which prompts help most
2. **Identify patterns**: When are you most/least motivated?
3. **Pre-decide rewards**: Have dopamine menu ready
4. **Stack motivations**: Combine multiple tactics
5. **Be specific**: "I need motivation" ‚Üí "I need energy matching"

### When Nothing Works
Sometimes low motivation signals:
- Need for rest (not laziness!)
- Wrong task for wrong time
- Underlying emotion to process
- Physical need (food, water, sleep)
- Task doesn't align with values

**Permission**: It's okay to rest or pivot!
