# AI Prompts - Task Initiation & Breaking Down Overwhelm

## Overview
These prompts help you start tasks when you feel stuck, overwhelmed, or can't figure out where to begin.

---

## Prompt 1: Task Breakdown Helper

### When to Use
- A task feels too big or overwhelming
- You know what needs to be done but don't know how to start
- You're stuck in planning paralysis

### The Prompt
```
I have a task that feels overwhelming: [TASK NAME]

Can you help me break this down into tiny, manageable steps? Please:
- Make each step take 15 minutes or less
- Start with the absolute easiest first step
- Use encouraging language
- Number the steps clearly

My current energy level is [1-10]: ___
My available time today: ___
Any constraints: ___
```

### Example Usage
```
I have a task that feels overwhelming: "Write annual performance review"

Can you help me break this down into tiny, manageable steps? Please:
- Make each step take 15 minutes or less
- Start with the absolute easiest first step
- Use encouraging language
- Number the steps clearly

My current energy level is [1-10]: 6
My available time today: 2 hours
Any constraints: Need to submit by Friday
```

### Copy-Ready Version (Minimal)
```
Break down this overwhelming task: [TASK]
- Energy level: [1-10]
- Available time: [AMOUNT]
- Make steps 15 min or less, start with easiest
```

## Prompt 2: Getting Started Nudge

### When to Use
- You can't make yourself begin
- Task initiation paralysis
- Need a gentle push

### The Prompt
```
I'm stuck starting this task: [TASK NAME]

I know I need to do it, but I can't seem to begin. Can you:
1. Give me just the first 2-minute action I need to take
2. Explain why starting is often the hardest part (help me feel less alone)
3. Suggest a reward for after I complete the first tiny step

Context about why this is hard: ___
```

### Example Usage
```
I'm stuck starting this task: Organizing my closet

I know I need to do it, but I can't seem to begin. Can you:
1. Give me just the first 2-minute action I need to take
2. Explain why starting is often the hardest part (help me feel less alone)
3. Suggest a reward for after I complete the first tiny step

Context about why this is hard: It feels like it will take forever and I don't know where to start
```

### Copy-Ready Version (Minimal)
```
Stuck starting: [TASK]
Give me just the first 2-minute action + why starting is hardest + reward suggestion
```

## Prompt 3: Task Analysis for ADHD Brain

### When to Use
- You're catastrophizing how hard something will be
- Need reality check on actual difficulty
- Want to identify potential obstacles

### The Prompt
```
I need to do: [TASK NAME]

Please analyze this task for my ADHD brain:
- What's the actual difficulty level? (sometimes I overestimate)
- What's the minimum viable version of this task?
- What distractions might pull me away?
- What environmental setup would help?
- What similar tasks have I done successfully before?
```

### Example Usage

I need to do: Respond to 20 work emails

Please analyze this task for my ADHD brain:

## Prompt 4: Overwhelm Reducer

### When to Use
- Everything feels urgent at once
- Brain dump has too many items
- Paralyzed by long to-do list

### The Prompt
```
I'm feeling overwhelmed by everything I need to do. Here's my list:
[PASTE YOUR BRAIN DUMP OR TO-DO LIST]

Please help me:
1. Identify what's ACTUALLY urgent vs what feels urgent
2. Find 1-3 quick wins (under 5 minutes each)
3. Suggest what I can postpone without guilt
4. Remind me that I can't do everything today, and that's okay

My energy level right now: [1-10]
Time available today: ___
```

### Example Usage
```
I'm feeling overwhelmed by everything I need to do. Here's my list:
- Write report
- Call dentist
- Buy groceries
- Respond to emails
- Fix printer
- Plan vacation
- Organize closet
- Schedule car maintenance
- Return library books
- Water plants

Please help me:
1. Identify what's ACTUALLY urgent vs what feels urgent
2. Find 1-3 quick wins (under 5 minutes each)
3. Suggest what I can postpone without guilt
4. Remind me that I can't do everything today, and that's okay

My energy level right now: 4/10
Time available today: 3 hours
```

### Copy-Ready Version (Minimal)

## Prompt 5: Decision Paralysis Breaker

### When to Use
- Stuck choosing between options
- Spinning on a decision
- Need external processing

### The Prompt
```
I'm stuck choosing between: [OPTION A] and [OPTION B]

My ADHD brain is spinning. Can you:
- List 3 pros and cons for each (brief)
- Suggest a simple "coin flip" test: which option makes me feel relieved?
- Remind me that most decisions aren't permanent
- Give me permission to just pick one and move on

Context: ___
Why this matters: ___
Deadline to decide: ___
```

### Example Usage
```
I'm stuck choosing between: Starting project A vs Starting project B

My ADHD brain is spinning. Can you:
- List 3 pros and cons for each (brief)
- Suggest a simple "coin flip" test: which option makes me feel relieved?
- Remind me that most decisions aren't permanent
- Give me permission to just pick one and move on

Context: Both are important, both have approaching deadlines
Why this matters: Can't do both at once, need to commit
Deadline to decide: Today
```

### Copy-Ready Version (Minimal)
```
Stuck deciding: [OPTION A] vs [OPTION B]
Give me: 3 pros/cons each, coin flip test, permission to just pick one
```

## Prompt 6: The "But I Don't Wanna" Helper

### When to Use
- Resistance to starting
- Emotional overwhelm about task
- Need validation plus strategy

### The Prompt
```
I really don't want to do this task: [TASK NAME]

I'm feeling [emotion: resistant/anxious/bored/overwhelmed/etc.]

Can you:
- Validate why this task might suck or be hard
- Help me identify the ACTUAL reason I'm avoiding it
- Suggest the smallest possible starting point
- Remind me it doesn't have to be perfect
- Give me permission to do just 10 minutes and stop

What I'm really avoiding: [guess if you can]
```

### Example Usage
```
I really don't want to do this task: Filing my taxes

I'm feeling anxious and overwhelmed

Can you:
- Validate why this task might suck or be hard
- Help me identify the ACTUAL reason I'm avoiding it
- Suggest the smallest possible starting point
- Remind me it doesn't have to be perfect
- Give me permission to do just 10 minutes and stop

What I'm really avoiding: Probably fear of making mistakes or owing money
```

### Copy-Ready Version (Minimal)
```
Don't wanna do: [TASK]
Feeling: [EMOTION]
Help me: validate it sucks, find real reason, smallest start, permission to stop after 10 min
```

## Prompt 8: Boring Task Gamification

### When to Use
- Task is necessary but incredibly dull
- Need to make it more engaging
- Struggling with understimulation

### The Prompt
```
I have to do this boring task: [TASK NAME]

It's necessary but my brain is not interested at all.

Help me gamify this:
- Suggest a challenge or competition (even with myself)
- Design a reward system
- Add sensory stimulation ideas (music, movement, etc.)
- Create artificial urgency or stakes
- Make it social somehow

Time I have: ___
Energy level: [1-10]
```

### Example Usage

- Need immediate action

## Prompt 10: Task Bundling Strategy

### When to Use
- Have many small related tasks
- Want to batch for efficiency
- Need to reduce context switching

### The Prompt
```
I have these related tasks: [LIST TASKS]

They're all [type: emails/calls/errands/etc.]

Help me:
- Group them logically
- Put them in optimal order
- Estimate total time (with ADHD buffer)
- Suggest when to take breaks
- Identify any I can delegate or delete

Goal: Get through these efficiently without burnout
```

### Example Usage
```
I have these related tasks:
- Call doctor for prescription
- Schedule dentist appointment
- Call insurance about bill
- Check in with mom
- Return call to friend
- Call mechanic about car

They're all phone calls

Help me:
- Group them logically
- Put them in optimal order
- Estimate total time (with ADHD buffer)
- Suggest when to take breaks
- Identify any I can delegate or delete

Goal: Get through these efficiently without burnout
```

### Copy-Ready Version (Minimal)
```
Tasks to bundle: [LIST]
All are: [TYPE]
Help me: group, order, time estimate + buffer, break schedule, delegation options
```

## How to Use These Prompts

### Quick Start
1. Find the prompt that matches your situation
2. Copy the "Copy-Ready Version" for speed
3. Fill in the blanks with your specifics
4. Paste into ChatGPT, Claude, or your AI assistant
5. Follow the AI's guidance

### Getting Better Results
- **Be specific**: More context = better help
- **State your energy level**: Helps AI calibrate suggestions
- **Mention constraints**: Time, resources, environment
- **Share why it's hard**: Helps AI address real obstacles
- **Ask for adjustments**: "Make it simpler" or "More detail please"

### Save Your Favorites
- Keep a note of prompts that work best for you
- Customize the wording to match your voice
- Create quick-access buttons in your Notion dashboard
